# HBVepsilon-MD
In this repository, post-processed Md data can be found from simulations of HBV epsilon RNA (PDBID=6VAR). Publication under review.
PCA script and data are labeled 'PCA'
Angle time dependent data and histogrammed data are labeled 'Angle'
2DRMS data is labeled '2DRMS' and presented as gnuplot files

All data is natively output from CPPTRAJ.
